
Для запуска многопользовательского чата запустите main-классы (в директории starters) в правильном порядке:
1.Для запуска сервера запустите ServerMain. В случае, если все прошло успешно, в консоли появится 'Сервер запущен!'
2.Для запуска консоли монитора чата запустите Monitor. В случае, если все прошло успешно,
в консоли появится приветсвенное сообщение
3.Для запуска консоли первого пользователя запустите ClientStarter. Перед запуском консоли, вы можете задать имя
пользователя в конструкторе класса Client. В случае, если все прошло успешно, в консоли появится поле ввода сообщения
Если вы хотите запусить двух и более пользователя, необходимо создать и запусть консоли пользователей
по подобию ClientStarter.

После отправки сообщения, вы можете его увидеть в консоли монитора чата Monitor.
Для обращения к боту небходимо начать сообщение со слова 'бот' без знаков препинания, например 'бот привет', ответ бота
появится в консоли монитора. Более подробно смотрите класс SimpleBot
